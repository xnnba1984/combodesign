# Data

`combination.csv` is the public PDX combination-treatment data file used by the
case-study script.

The case-study code expects the file at:

```text
data/combination.csv
```

Before making a public repository, confirm that redistributing this raw data
file is acceptable under the original source terms and your institution's
policy. If redistribution is not allowed, remove the CSV and provide download
instructions that restore the same local file path.
